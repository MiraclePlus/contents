## 关于 MiraclePlus 奇绩创坛 [跳转官网](https://www.miracleplus.com/?p=ba&c=ib&utm_source=github) · [立即申请](https://apply.miracleplus.com/?p=ba&c=ib&utm_source=github)


奇绩创坛提供中国早期创业的加速服务，帮助初创公司从 0 到 1，做到最关键的产品与市场匹配（PMF）。

创业营每年举办 2 次，奇绩创坛会投资加速一大批初创团队，每家 30 万美元（或等值人民币）换取 7% 的公司股份（股权投资）。这笔投资代表了我们和创始人站在一起的决心，也是加强创始人与校友社区连接的有效手段。投资后，合伙人会像共同创始人一样高强度地和创始人们一起工作 3 个月，让每个团队快速迭代到最好的状态，在路演日（每期有 1000+ 位投资人到场）迎接上千位投资人。

- **股权投资**：30 万美元（或等值人民币）换取 7% 股份，与创始人站在一起。
- **合伙人指导**：合伙人每周通过 office hour 与创始人一对一讨论遇到的问题与困难，平均每个项目获得约 100 小时一对一辅导。
- **奇绩社区**：与全球 1000+ 来自前沿领域的早期创业者一起创业，通过 group office hour 与同行业、上下游创业者交流。
- **路演日**：在舞台上向 1000+ 位投资人有说服力地呈现公司产品与服务，大幅提升融资效率。
- **闭门分享**：创业营期间每隔一周邀请知名企业家（如张一鸣、雷军、周鸿祎、丁磊、李想等）与创始人面对面分享交流。
- **专业团队**：合伙人团队包括陆奇博士、栾运明、曹勖文、毛圣博等不同技术和创业领域的专业人士。
